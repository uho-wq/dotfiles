# プロテイン選びガイド（MyProtein 中心）

## ホエイプロテイン vs クリアホエイ

| 項目 | Impact Whey Protein（通常ホエイ） | Clear Whey Isolate（クリアホエイ） |
|---|---|---|
| 原料 | WPC（ホエイ濃縮物）中心 | WPI（ホエイ分離物）＝高純度 |
| 見た目・食感 | 牛乳やミルクシェイクのように濃厚・不透明 | ジュースのように透明でサラッと |
| 1食あたりタンパク質 | 約 21g | 約 20g |
| 脂質・炭水化物 | 少量含まれる（1〜2g程度） | ほぼゼロ |
| 乳糖 | 含まれる（お腹がゆるくなる人も） | ごく微量（乳糖不耐性でも飲みやすい） |
| フレーバー傾向 | チョコ・バニラ・抹茶など「甘い系」 | 桃・ピーチティー・グレープ等「フルーツ系」 |
| 価格 | 安い（コスパ◎） | やや高め |
| 向いているシーン | 朝食代わり・間食・バルク期 | 運動中〜直後・夏場・食欲がない時 |

**ざっくり覚え方**
- ミルクシェイク感覚で栄養補給したい → ホエイ
- ジュース感覚でさっぱり飲みたい → クリア

## プロテインの選び方の基準

### 1. 目的で選ぶ
- **筋肥大・増量** → Impact Whey（カロリー・栄養が乗る）
- **減量・ダイエット** → Clear Whey または Impact Whey Isolate（脂質・糖質オフ）
- **健康・置き換え** → Impact Whey（価格と栄養のバランス◎）

### 2. 体質で選ぶ
- **乳糖でお腹を壊す** → Clear Whey / Isolate / ビーガンプロテイン
- **牛乳アレルギー・ビーガン** → ソイ、ピー、ブラウンライス等の植物性

### 3. 飲み心地の好みで選ぶ
- **濃厚・甘いのが好き** → Impact Whey（ミルキーチョコやナチュラルチョコが定番）
- **さっぱり系・夏でもガブ飲みしたい** → Clear Whey（ピーチティー、モヒート、オレンジマンゴーなど人気）

### 4. タイミングで選ぶ
- **朝・間食・就寝前** → Impact Whey（腹持ち◎）
- **トレーニング中〜直後** → Clear Whey（吸収が速く、胃にもたれない）
- **寝る前にゆっくり吸収** → カゼイン（Slow-Release Casein）

### 5. コストで選ぶ
- **最安・続けやすさ重視** → Impact Whey（セール時 1kg 2,000円台も）
- **多少高くても飲み心地優先** → Clear Whey

### 6. フレーバー選びのコツ
- 初めてなら**定番のチョコ系**（ナチュラルチョコ、ミルクティー、抹茶）が無難
- クリアは**ピーチティー、マンゴー、ラズベリー**が日本人に人気
- 大きいサイズでいきなり買わず、**まず 250g〜1kg で試す**のが鉄則

## 「Impact Whey 甘すぎ」問題への対処

### 甘さの実情
- 人工甘味料（**スクラロース**）がガッツリ効いていて、日本の市販プロテインよりかなり甘い
- 特に海外フレーバー（Cookies & Cream、Chocolate Brownie など）は後味までしっかり甘い
- 水で溶かすと甘さが際立ち、牛乳で溶かすとマイルドになる

### 甘さ控えめで失敗しにくいフレーバー

| フレーバー | 甘さレベル | コメント |
|---|---|---|
| **ナチュラルチョコレート** | ★★☆☆☆ | 甘さ控えめ、ビターよりのココア感。日本人に一番人気 |
| **ナチュラルバニラ** | ★★☆☆☆ | 素朴で飲みやすい、牛乳と相性◎ |
| **抹茶ラテ** | ★★☆☆☆ | 抹茶の苦味で甘さが中和される、日本人向け |
| **ミルクティー** | ★★★☆☆ | 紅茶の風味で甘さが気にならない、定番人気 |
| **ティラミス** | ★★★☆☆ | コーヒー感で甘さ緩和 |
| **ナチュラルストロベリー** | ★★★☆☆ | 甘いがフルーツ系で飲める |
| Chocolate Smooth | ★★★★☆ | しっかり甘い |
| Cookies & Cream | ★★★★★ | かなり甘い、アメリカ系 |
| Chocolate Brownie | ★★★★★ | デザート級の甘さ |

→ **「ナチュラル」と付くフレーバーは甘さ控えめ**。迷ったらまずここ。

### 甘さを抑える飲み方

1. **水の量を増やす**（規定 250ml → 350〜400ml に）
2. **牛乳・豆乳で割る**（甘さマイルド、こってり感）
3. **1 食分を 2 回に分けて飲む**（スクープ半分ずつ）
4. **氷を入れてシェイク**（冷たいほど甘さを感じにくい）
5. **無糖ヨーグルト・オートミールに混ぜる**（食事化して甘さを分散）
6. **ブラックコーヒーや無糖アーモンドミルクで割る**（大人の味に）

### 甘いのが本当に苦手なら

- **Impact Whey Isolate のノンフレーバー（味なし）** → 完全に無味・無臭。自分で調整できる上級者向け
- **Clear Whey** → ジュース系なので「甘ったるさ」は少なめ
- **ビーレジェンド、ULTORA など国産プロテイン** → 甘さ控えめ設計が多いが単価は高い

## 最終的な指針

- **迷ったら** → Impact Whey の「ナチュラルチョコレート」か「ミルクティー」1kg
- **ダイエット中・夏・お腹弱め** → Clear Whey の「ピーチティー」500g
- **ビーガン・乳アレ** → Soy Protein Isolate または Vegan Blend
- **甘さ完全コントロール派** → Impact Whey Isolate のノンフレーバー
